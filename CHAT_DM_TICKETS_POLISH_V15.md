# Chat + Tickets Polish V15

## DM fixes
- Reaction picker now opens only when the actual message bubble is clicked, not when clicking empty row space.
- Message action menu is anchored directly to the message bubble instead of a detached fixed viewport position.
- Reply/Edit/Pin/Delete menu uses compact modern popover styling and stays close to the selected message.

## Tickets UI improvements
- Reduced KPI card height and spacing.
- Converted filters into a compact futuristic toolbar.
- Priority/type selects now use fixed compact widths.
- Ticket search no longer consumes the full row unnecessarily.
- Kept status chips, My Tickets, priority, type, and search in a cleaner single control strip.
